<h3>#Requirements</h3>
<p> Node version > 16 </p>
<p> npm > 8 </p>
<p> (Android or Ios emulator with SDK for Android) or Expo Go App on Android or Ios device </p>

<h3>#Installation</h3>
<p>git clone git@github.com:rupesh-10/greenguru.git </p>
<p>cd greenguru </p>
<p> npm install </p>
<p> npm run start </p>
